## [簡易鍵盤錄製器](https://v2.tauri.app/plugin/file-system/)

## [安裝執行](https://william-weng.github.io/tags/rust/)
```bash
brew install node
brew install rust
npm install
npm run tauri build
```

## 快速鍵
- CONTROL + SHIFT + S => 開始 / 停止錄製
- CONTROL + SHIFT + P => 執行回放
